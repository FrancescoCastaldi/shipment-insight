# Shipment Insight

> A lightweight, client-side shipment tracking dashboard for GLS parcels. Fully static — no build step, no framework, no dependencies.

![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-blue)
![Static](https://img.shields.io/badge/static-HTML%2FCSS%2FJS-green)

## Features

- **Shipment Dashboard** — View all your shipments in a clean table layout
- **Add & Track** — Add new shipments with tracking numbers and courier info
- **One-Click Tracking** — Open the courier's tracking page directly from the dashboard
- **GLS Support Ticket** — Quick access to the GLS ServiceNow portal
- **Email Assistant** — Pre-formatted email templates for requesting shipment updates
- **Local-First** — All data stored in your browser's localStorage (no server needed)

## Usage

Simply open `index.html` in your browser. No installation required.

Live demo: [https://francescocastaldi.github.io/shipment-insight/](https://francescocastaldi.github.io/shipment-insight/)
